# Trigger2to3
VRChat SDK3でVRC_Triggerを使いたい人のための代替ツールです。  
VRC_Triggerの機能の大部分を取り込み、VRC_Triggerに近い使用感でワールドギミックを組み立てることができます。  
VRC_TriggerのUIを次のような3つのUdonで表現して、実現します。
